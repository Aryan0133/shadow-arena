## 📱 Project Overview

**Shadow Arena** is a fast-paced 2D survival action game for Android where the player navigates an arena, avoids enemies, collects power-ups, and survives as long as possible.

This project demonstrates real-world **Android game development**, **real-time rendering**, and **modular OOP architecture**.

---

## 🚀 Features

- 🎮 Touch-based player movement
- 🤖 Enemy AI (chasing behavior)
- ⚡ Dynamic difficulty scaling
- 🔫 Expandable shooting system
- ❤️ Player health system
- 🏆 Survival-based scoring
- 🎨 Material Design UI
- 🎵 Audio using MediaPlayer
- ⚙️ SurfaceView game loop (~60 FPS)
